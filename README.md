# Healthcare Analytics Dashboard using Tableau

## 📊 Project Overview

Healthcare organizations generate massive amounts of data related to patient demographics, diseases, treatments, and hospital operations. Analyzing this data effectively can help healthcare providers improve decision-making, optimize resources, and enhance patient care.

This project focuses on building an **interactive healthcare analytics dashboard using Tableau** to visualize and analyze patient data. The dashboard provides insights into patient demographics, disease distribution, departmental workload, and treatment costs.

The goal of this project is to transform raw healthcare data into meaningful visual insights that can support healthcare administrators in data-driven decision making.

---

## 🎯 Objectives

* Analyze patient demographic information
* Identify patterns in disease distribution
* Visualize department-wise patient statistics
* Understand treatment cost trends
* Build an interactive dashboard for healthcare analytics

---

## 🛠️ Tools & Technologies

* **Tableau**
* **Microsoft Excel / CSV Dataset**
* **Data Visualization Techniques**

---

## 📊 Dashboard Features

The dashboard provides several analytical visualizations including:

* Patient gender distribution
* Age group analysis
* Department-wise patient count
* Disease frequency analysis
* Treatment cost analysis
* Admission trends

These visualizations help healthcare managers quickly understand operational patterns.

## ▶️ How to Run the Project

1. Install **Tableau Desktop**
2. Download this repository
3. Open the file:

```
healthcare_dashboard.twb
```

4. Connect the dataset if prompted
5. Explore the interactive dashboard

## 📈 Key Insights
